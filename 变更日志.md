# 变更日志

## [1.2.0] - 2025-12-25

### 已添加
- 实现商家销售统计功能，商家中心现在可以显示销售数据（总订单数、总销售额、平均订单金额）
- 添加商品审核功能，管理员可以批准待上架的商品
- 在管理员面板中添加"待审核商品"选项卡，便于管理待审核商品

### 已更改
- 更新商品状态显示逻辑，将inactive状态显示为"待审核"，使流程更清晰
- 在商家创建商品后显示审核流程提示信息
- 更新管理员仪表板，添加待审核商品统计
- 更新前端各组件中的商品状态描述，确保一致性
- 更新数据库初始化脚本，新商品默认状态设为'inactive'（待审核状态）
- 更新数据库迁移脚本，确保新商品默认状态为'inactive'（待审核状态）

### 已修复
- 修复商家中心无法显示收益数据的问题
- 修复商品审核流程缺失的问题

### 文档更新
- 更新API接口文档，添加商品审核相关接口
- 更新用户故事文档，添加商品审核功能说明

## [1.1.0] - 2025-12-25

### 已更改
- 重构项目结构，分离前后端代码
- 创建清晰的目录结构，便于维护
- 添加文档、数据库和Docker配置目录
- 更新项目说明.md文件以反映新结构
- 将所有文档文件重命名为中文名称，便于理解

### 目录结构变更
- 旧结构: 项目根目录下混合前后端代码
- 新结构:
  - backend/: 后端 Node.js + Express 代码
  - frontend/: 前端 Vue3 + Vite 代码
  - docs/: 项目文档
  - database/: 数据库相关文件
  - docker/: Docker 配置文件

## [1.0.0] - 2025-12-24

### 已发布
- 初始版本的简易在线商城系统
- 实现买家、商家、管理员三种角色功能
- 完整的前后端分离架构
