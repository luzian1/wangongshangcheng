# 前端项目说明

## 项目概述

这是一个基于 Vue3 + Vite 的前端项目，为皖工购物网系统提供用户界面。项目实现了买家、商家、管理员三种角色的专门界面，确保不同角色用户有明确且专门的体验。

## 技术栈

- **前端框架**: Vue3
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **样式处理**: Sass
- **HTTP 客户端**: Axios

## 项目结构

```
frontend/
├── public/                 # 静态资源
├── src/
│   ├── assets/             # 静态资源文件
│   ├── components/         # Vue 公共组件
│   ├── views/              # 页面视图组件
│   ├── router/             # 路由配置
│   ├── stores/             # Pinia 状态管理
│   ├── services/           # API 服务封装
│   ├── styles/             # 全局样式文件
│   ├── utils/              # 工具函数
│   ├── App.vue             # 根组件
│   └── main.js             # 入口文件
├── package.json            # 项目配置
└── vite.config.js          # Vite 配置
```

## 快速开始

### 环境要求
- Node.js v16 或更高版本
- npm 或 yarn

### 安装与运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```

## 角色界面分离

本项目实现了不同角色的专门界面：

- **买家**: 默认首页，显示商品列表和购物功能
- **商家**: 专门的商家中心，显示商品管理功能
- **管理员**: 专门的管理控制台，显示系统监控和管理功能

## API 集成

前端通过 `services/` 目录下的服务文件与后端 API 进行交互，所有 API 调用都经过统一的错误处理和认证管理。

## 状态管理

使用 Pinia 进行全局状态管理，包括：
- 用户认证状态
- 购物车数据
- 商品数据
- 订单数据
